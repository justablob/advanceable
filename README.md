# Advanceable

Advanceable is a simple Node.js Buffer wrapper that allows reading of a Buffer using an internal incrementing offset.

### Installation

Advanceable is available on npm.

```shell

# with npm:
npm install advanceable

# with yarn:
yarn add advanceable

```
